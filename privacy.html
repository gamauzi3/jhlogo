Johanna 개인정보처리방침

최종 수정일: 2026년 3월 30일

1. 수집하는 개인정보

본 서비스는 Google Workspace 계정을 통해 다음 정보를 수집합니다.

- Google Workspace 이메일 주소
- Google Workspace 표시 이름
- Google Chat 대화 내용 (업무 질의 내용)
- Asana 연동 시: Asana OAuth 토큰

2. 개인정보 이용 목적

수집된 정보는 다음 목적으로만 사용됩니다.

- 사용자 식별 및 맞춤형 업무 응답 제공
- Asana, ERP, Google Drive 등 연동 서비스 인증
- 서비스 품질 개선 및 오류 수정

3. 개인정보 보관 및 파기

대화 기록은 30분간 세션 유지 후 자동 삭제됩니다. Asana OAuth 토큰은 암호화되어 내부 서버에 저장되며, 사용자 요청 시 즉시 삭제합니다.

4. 개인정보의 제3자 제공

수집된 개인정보는 제3자에게 제공되지 않습니다. 모든 데이터는 회사 내부 서버에서만 처리됩니다.

5. 외부 서비스 연동

본 서비스는 다음 외부 API와 연동되며, 각 서비스의 개인정보처리방침이 적용됩니다.

- Google Workspace API (Calendar, Chat, Drive, Gmail)
- Asana API
- Google Gemini API (AI 응답 생성)

AI 응답 생성 시 사용자의 질문이 Google Gemini API로 전송되며, Google의 API 이용약관이 적용됩니다.

6. 정보주체의 권리

사용자는 언제든지 자신의 데이터 삭제를 요청할 수 있으며, Asana 연동 해제를 요청할 수 있습니다.
